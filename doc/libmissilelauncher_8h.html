int16_t ml_reference_launcher ( ml_launcher_t *  launcher)

Refrences a launcher so that it isn't destroyed. If a launcher isn't refrenced, when you call ml_free_launcher_array the refrence may or may not be freed. Thus you can't guarantee that the memory is still there for you to use. Using ml_refrence_launcher with ml_dereference_launcher avoids this issue.

Parameters
launcherThe launcher to reference.
Returns
A status code
